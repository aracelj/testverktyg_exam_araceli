# User Stories
By Araceli Jakobsson

[U1] As a user, I want to click the heart icon that appears beside a selected book so that I can mark the book as a favorite.

[U2] As a user, I want to open the “Add New Book” tab so that I can add a new book with a title and author to the catalog list.

[U3] As a user, I want to open the “My Books” tab so that I can view my list of favorite books.

[U4] As a user, I want to open the "Statistics" tab so that I can see how many books I have marked as favorites.
